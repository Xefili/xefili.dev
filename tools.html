Tools

Here is an overview of all the tools and programs I use

Visual Studio Code

I use Visual Studio Code to write the main code for the website.

For TailwindCSS support and IntelliSense, I also use their extension which really helps finding the correct utility class names.

TailwindCSS

All styling is done through TailwindCSS. This simplifies development and reduces massive CSS files. Okay, the CSS file is massive, but it could very well be even more massive.

As for the little animations on the main page, they are plain CSS animations.

Github

To keep track of changes, I use Github. It integrates great with Visual Studio Code and makes it easy to keep the site current, since the Cloudflare Workers & Pages system keeps rebuilding my Tailwind files every time there is a push to the repo.

Cloudflare

Domain Management and Registration is done with Cloudflare. They offer all-in-one DNS management and tunneling services with their Zero Trust platform. Cloudflare Tunnels connect the API to the world without me having to expose my non-static IP address to the internet.

The website is also hosted by CLoudflare with CF Workers & Pages. It continously pulls new files from the Github repo and rebuilds the CSS file. All without any human intervention!
