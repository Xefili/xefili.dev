Main Page - in detail

The Head

In this section, we are going to dive into the structure and design decisions of the main page @ xefili.dev
The following snippet shows the <head> element of the page, containing links and refrences to important files which are required on pageload

<head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <!--Link Tags-->
        <link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Material+Symbols+Outlined:opsz,wght,FILL,GRAD@48,400,1,0" />
        <link rel="preconnect" href="https://fonts.googleapis.com">
        <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
        <link href="https://fonts.googleapis.com/css2?family=Poppins:wght@400;700;900&family=Roboto+Flex:opsz,wdth,wght@8..144,112.5,400;8..144,112.5,700;8..144,112.5,900&display=swap" rel="stylesheet">
        <!-- <link id="swap" rel="stylesheet" href="styles/style.css"/> -->
        <link id="swap" rel="stylesheet" href="styles/build.css"/>
        <link rel="icon" type="image/x-icon" href="assets/Bucket Helper.svg">
        <title>Home - Bucket Lounge</title>
    </head>

Line 2 defines the character set of the HTML document, for compatability reasons, this is set to UTF-8 by default. Line 3 prevents mobile devices from spoofing their width. These two lines are in every single HTML header by default.

The lines 5 to 8 import Google fonts - the first imports the material symbols font, used for the icons in the navigation bar whereas line 8 imports the font Poppins, used for titles.

The most important file here is linked on line 10: the CSS file. This file contains all used CSS classes generated by Tailwind. Without this file, the page would look really bad and would have no structure at all. Line 11 links the little icon and line 12 sets the title of the page
